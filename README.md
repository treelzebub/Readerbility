# Readerbility

Aggregator:
1) Login to Readability
  a) directly
  b) social login
2) pull bookmarked Articles
  a) search
  b) browse tags
  c) sort by *
3) select any number of Articles and collect each as an HTML file within an epub, mobi, [et al.] file.
4) save to cloud
5) open in your favorite e-reader.
